# ChronoView

Headless schedule timeline library for React/Next.js.

## Packages

| Package | Description |
| --- | --- |
| `@chronoview/core` | Framework-agnostic core engine |
| `@chronoview/react` | React hooks and context (headless) |
| `@chronoview/ui` | shadcn-style UI components |

## Development

```bash
pnpm install
pnpm build
pnpm test
pnpm lint
```

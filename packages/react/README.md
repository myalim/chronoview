# @chronoview/react

React hooks and context for [ChronoView](https://github.com/myalim/chronoview) — headless schedule timeline components.

Use this package for full control over rendering. For ready-to-use components, install `@chronoview/ui` instead.

See the [main README](https://github.com/myalim/chronoview#readme) for full documentation.

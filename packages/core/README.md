# @chronoview/core

Framework-agnostic core engine for [ChronoView](https://github.com/myalim/chronoview) — time calculations, event positioning, stacking algorithms, and virtual scroll.

This package is used internally by `@chronoview/react` and `@chronoview/ui`. Most users should install `@chronoview/ui` directly.

See the [main README](https://github.com/myalim/chronoview#readme) for full documentation.

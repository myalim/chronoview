# @chronoview/core

## 0.1.0-alpha.0

### Minor Changes

- Initial alpha release of ChronoView.

  - Core engine: time calculations, event positioning, stacking algorithms, virtual scroll
  - React hooks: useScheduleView, useDateNavigation, useNowIndicator, useVirtualScroll, useTimelineFilter
  - UI components: Schedule (Day/Week/Month), EventCard, TimeHeader, ResourceSidebar, NowIndicator, GridLines
  - Common components: Button, DateNavigator, ViewToggle, Toolbar, FilterChips, CategoryTabs
  - Pre-bundled CSS (Tailwind utilities + design tokens)
  - Dark mode support

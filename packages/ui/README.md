# @chronoview/ui

Pre-built UI components for [ChronoView](https://github.com/myalim/chronoview) — Schedule, Calendar, and Grid views with design tokens and dark mode.

## Install

```bash
pnpm add @chronoview/ui
```

## Setup

Import the stylesheet:

```css
@import "@chronoview/ui/styles.css";
```

## Usage

```tsx
import { Schedule } from "@chronoview/ui";

<Schedule events={events} resources={resources} />
```

See the [main README](https://github.com/myalim/chronoview#readme) for full documentation.
